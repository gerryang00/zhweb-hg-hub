# zhweb-hg-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自独立，不针对任何特定域名或网站，仅作为通用页面存档使用。

## 项目简介

zhweb-hg-hub 是一个静态 HTML 页面集合仓库。它收录了多个单独编写的 HTML 文档，涵盖不同用途和场景。每个页面都是独立的，不依赖外部资源或特定服务器环境。

## 目录说明

仓库根目录下包含若干 `.html` 文件，每个文件对应一个独立页面。目录结构保持扁平，方便直接访问和引用。

- `index.html` — 仓库首页（若有）
- 其他 `.html` 文件 — 各个独立页面

如需添加新页面，请直接将 HTML 文件放入仓库根目录。

## 页面归档说明

所有页面均为纯静态 HTML，不包含后端逻辑。归档标准包括：

- 页面结构完整，包含基本 HTML 标签和元信息
- 可独立在浏览器中打开并正常显示
- 不包含外部链接或引用（除非是通用 CDN 资源）
- 不涉及任何具体域名、品牌或商业内容

## 维护说明

本仓库由维护者不定期更新。页面内容可能根据需要进行调整或替换。

- 欢迎提交 Issue 或 Pull Request 建议新增页面
- 提交的页面应保持独立性和通用性
- 请勿提交包含具体域名、营销信息或侵权内容的文件

## 使用方式

您可以直接通过浏览器打开仓库中的任意 `.html` 文件，或将其部署到任何支持静态文件的服务器上。

## 许可证

本仓库内容以通用方式提供，具体使用请参考仓库内的 LICENSE 文件（如有）。
